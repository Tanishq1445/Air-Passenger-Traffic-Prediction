# Air Passenger Traffic Prediction using Machine Learning

## Overview
This project focuses on forecasting monthly air passenger traffic using historical aviation data and machine learning techniques.

## Features
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Passenger traffic trend visualization
- Regression model comparison
- Future passenger traffic prediction

## Models Used
- Linear Regression
- Random Forest Regression
- Gradient Boosting Regression

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset
Air Traffic Passenger Statistics Dataset from Data.gov

## Results
Random Forest Regression achieved the best prediction performance among all trained models.
